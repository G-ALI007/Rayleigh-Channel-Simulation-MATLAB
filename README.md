📡 Rayleigh Fading Channel Simulation
🌐 Overview
This MATLAB project simulates the Bit Error Rate (BER) performance over a Rayleigh Fading Channel. It compares three cases:

Uncoded Transmission.

Coded without Interleaving.

Coded with Interleaving.

🛠️ Simulation Features
Modulation: BPSK.

Error Correction: Hamming (7,4).

Technique: Random Interleaving.

Channel: Multipath Rayleigh Fading with Doppler shift.

🚀 How to Run
Open the file Fading_Channel_Interleaving.m in MATLAB.

Run the script to see the BER performance plots.
